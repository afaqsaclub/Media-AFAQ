## 🛠️ كيفية الاستخدام

### للمستخدمين العاديين
1. افتح الملف `afaq_website.html` في المتصفح
2. ستظهر شاشة تسجيل الدخول
3. أدخل اسم المستخدم وكلمة المرور من الجدول أعلاه (يمكن استخدام النسخة الكاملة AFAQ_xxx أو الاختصار)
4. اضغط "دخول"
5. ستنتقل مباشرة إلى الموقع
6. انتقل إلى قسم "المصرح لهم" من القائمة لتعرض جدول المستخدمين المعتمدين
7. اضغط على اسمك في أعلى يسار الشاشة لتسجيل الخروج

### للمطورين - إضافة مستخدم جديد
1. افتح الملف `afaq_website.html`
2. ابحث عن `USERS_DB`
3. أضف مستخدم جديد بالصيغة:
```javascript
'username': { 
  password: 'password123', 
  name: 'اسم المستخدم', 
  role: 'member', 
  avatar: 'emoji' 
},
```
