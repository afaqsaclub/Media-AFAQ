```

الحل:   
1. تأكد من اسم الملف: afaq_website.html
2. جرّب متصفح مختلف
3. تحميل الملف مجدداً
```

### رسالة خطأ عند الدخول
```
السبب:   اسم المستخدم أو كلمة المرور خاطئة
الحل:
1. غيّر حالة الحروف (CAPS LOCK)
2. استخدم أحد الحسابات من الأعلى
3. امسح المسافات الزائدة
```

```
السبب:   localStorage معطّل
الحل:
1. افعّل localStorage في الإعدادات
2. جرّب متصفح آخر
3. امسح ذاكرة الالتخزين المؤقتة
```

```
السبب:   الإنترنت بطيء أو جهاز ضعيف

1. تحقّق من سرعة الإنترنت
```
