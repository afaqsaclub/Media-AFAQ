## ✅ خطوات الاختبار الأساسية

### 1. اختبار تسجيل الدخول
```
✓ افتح الملف afaq_website.html
✓ جرّب تسجيل دخول خاطئ → يجب أن ترى رسالة خطأ حمراء
✓ جرّب الحساب: admin / admin123 → يجب أن تدخل الموقع
✓ جرّب استخدام الأسماء الرسمية الكامل (AFAQ_Hussain_4827193 مثلاً)
✓ سترى اسمك في أعلى يسار الشاشة
```
### 2. اختبار التنقل
```
✓ جرّب الضغط على جميع الأقسام من الشريط الجانبي
✓ ضمن القائمة ستجد قسم "المصرح لهم" الذي يعرض جدول الأشخاص المسموح لهم
✓ على الجوال: اضغط ≡ (الهامبرغر) لفتح القائمة
✓ جرّب التنقل من أسفل الشاشة (على الجوال)
```

### 3. اختبار المناسبات
```
✓ انتقل إلى قسم "المناسبات"
✓ يجب أن ترى عداد حي لكل مناسبة
✓ جرّب تحديث الصفحة → العداد يستمر من حيث توقف
```

### 4. اختبار الجلسة
```
✓ سجّل دخول ثم أغلق المتصفح
✓ افتح الملف مرة أخرى
✓ يجب أن تكون مسجلاً دخولاً بالفعل (جلسة محفوظة)
```

### 5. اختبار تسجيل الخروج
```
✓ اضغط على اسمك في أعلى يسار الشاشة
✓ يجب أن تعود إلى شاشة الدخول
✓ يجب أن تُمسح البيانات المحفوظة
```

## 🚨 استكشاف الأخطاء

### المشكلة: الموقع لا يعمل
**الحل:** تأكد من أن JavaScript مفعّل في المتصفح

### المشكلة: رسالة خطأ "بيانات غير صحيحة"
**الحل:** استخدم أحد الحسابات المعروضة أعلاه

### المشكلة: الموقع بطيء جداً
**الحل:** قلّل تفاصيل الصور أو استخدم متصفح أحدث

### المشكلة: الجلسة لا تُحفظ
**الحل:** تأكد من أن localStorage غير معطّل في المتصفح
